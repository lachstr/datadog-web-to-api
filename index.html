Datadog: web-to-api

Instantly convert json widget definitions from the datadog webapp for the datadog api

Intended for use with dddk when porting custom widgets from the web app. Currently it wipes all information except requests and markers. Users are encouraged to add a relevant "style" property to each request. Forked from the project curl-to-go on github.
